# 🚀 À propos de moi

Futur ingénieur en Génie Informatique à l'**ENSA Oujda**, spécialisée en **Intelligence Artificielle**. 

Je me passionne pour la conception et le déploiement d'applications distribuées, résilientes et Cloud-Native. Mon approche technique se situe à l'intersection du Génie Logiciel, des architectures IA scalables et des pratiques DevOps/GitOps.

📍 Oujda, Maroc

---

### 🛠️ Travaux actuels & Objectifs
* **Projet phare :** Architecture et développement de **Talk2Job** — une plateforme de recrutement pilotée par l'IA, s'appuyant sur une architecture microservices (ASP.NET Core), des services de deep learning (Python/FastAPI) et un routage via API Gateway.
* **Expertises en cours de renforcement :** Architectures distribuées complexes, orchestration avancée avec Kubernetes, Infrastructure as Code (Terraform) et cycle de vie MLOps.
* **Collaborations recherchées :** Projets Open-Source innovants axés sur le Cloud Computing, l'écosystème Full-Stack et l'automatisation des infrastructures.
